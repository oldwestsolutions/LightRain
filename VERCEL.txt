LightRain on Vercel (one Git repo, three subdomains)

1) Create THREE Vercel projects, all linked to this repo.

2) For each project, set Root Directory (Settings → General → Root Directory):
   - Marketing:  apps/web
   - Wallet:     apps/wallet
   - Admin:      apps/admin

   Leave Install Command / Build Command empty so each app’s vercel.json runs:
   "cd ../.. && npm install" and "cd ../.. && npm run build -w @lightrain/…"

3) If you MUST leave Root Directory as "." (repo root) on a project:
   - Add env var LIGHT_RAIN_VERCEL_APP = web | wallet | admin (match the app).
   - Install: npm install
   - Build: npm run build
   Prefer Root Directory = apps/* instead; it fixes Next.js detection and output.

4) Domains (Settings → Domains):
   - web    → lightra.in / www
   - wallet → wallet.lightra.in
   - admin  → admin.lightra.in
